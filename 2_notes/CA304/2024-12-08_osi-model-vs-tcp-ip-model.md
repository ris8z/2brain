# Osi model vs Tcp/ip model

## Osi vs Tcp/ip
| **OSI Model**        | **TCP/IP Model**   |
|-----------------------|--------------------|
| Application           | Application        |
| Presentation          | Application        |
| Session               | Application        |
| Transport             | Transport          |
| Network               | Network            |
| Data Link             | Access             |
| Physical              | Access             |


## Tcp/ip protocols
| **TCP/IP Model Layer** | **Examples**        |
|-------------------------|---------------------|
| Application             | FTP, DNS, HTTP     |
| Transport               | TCP, UDP           |
| Network/Internet        | IP                 |
| Access                  | Ethernet, 802.11   |
